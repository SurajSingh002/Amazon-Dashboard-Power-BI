# 📌 Project Overview

Dataset file link = https://www.kaggle.com/datasets/surajsingh83/amazon-fashon-csv

This project focuses on building an interactive *Amazon Sales Dashboard* using *Power BI* to analyze sales performance, customer behavior, product trends, and business insights.

The dashboard provides a comprehensive visualization of Amazon sales data, helping businesses understand key performance indicators (KPIs) and make data-driven decisions.
